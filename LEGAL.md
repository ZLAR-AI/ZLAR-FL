## What ZLAR-FL Is

ZLAR-FL is an observability tool. It reads agent data from local files and reports on fleet status, health, and audit events. It does not control agents, modify their behaviour, or transmit data off the host machine.

## What ZLAR-FL Does Not Do

ZLAR-FL does not:
- Control any agent
- Prevent fleet-wide compromise
- Guarantee fleet health
- Manage agent behaviour
- Substitute for enterprise fleet management, SIEM, or security operations platforms

Fleet registry entries contain local file paths and hostnames. Do not store secrets or credentials in agent metadata. The registry is not encrypted.
